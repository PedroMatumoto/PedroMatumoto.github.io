# Minimalist Portfolio

A modern, minimalist portfolio website built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- **Minimalist Design**: Clean layout with plenty of whitespace.
- **Japanese Aesthetics**: Vertical text elements and Noto Serif JP font.
- **Animations**: Smooth reveal animations using Framer Motion.
- **Responsive**: Fully responsive design for all devices.

## Customization

To update the content with your own information:

1.  Open \src/data/portfolio.ts\.
2.  Update the \portfolioData\ object with your details:
    -   \
ame\: Your name.
    -   \	itle\: Your job title.
    -   \	itleJP\: Your job title in Japanese.
    -   \bout\: A short bio.
    -   \xperiences\: Add your work experience.
    -   \projects\: Add your projects.
    -   \contact\: Update your contact links.
